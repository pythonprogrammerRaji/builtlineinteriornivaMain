# Builtline Interior Nivas Website

## Overview

Builtline Interior Nivas is a professional interior design and construction business website developed to showcase interior services, design ideas, and customer engagement for residential and commercial spaces in Bangalore. The website is designed with a modern, responsive interface and includes SEO-focused service pages to improve online visibility and lead generation.

The project highlights various interior design solutions including living room, bedroom, modular kitchen, pooja room, dining room, and commercial interiors. It also includes a contact form for customer inquiries and blog-style content pages for interior design inspiration.

---

## Features

* Responsive and modern UI design
* SEO-optimized interior service pages
* Interior design blog section with dynamic slug-based content
* Service-based category filtering
* Search functionality for keywords and services
* Contact form integration using EmailJS
* Mobile-friendly navigation menu
* WhatsApp, phone, and map integration for customer contact
* Smooth animations using AOS (Animate On Scroll)
* Bootstrap-based responsive layout
